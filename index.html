⚡ 핵심 요약

  • ▶ 복수노조 시 교섭대표 선출은 법적 의무
  • ▶ 과반수 노조가 없으면 공동교섭대표단 구성
  • ▶ 소수노조도 공정대표의무로 보호받음

1. 교섭창구 단일화란?

하나의 사업장에 2개 이상의 노조가 있을 때, 교섭대표를 하나로 통일하는 법적 의무입니다.

📌 핵심 원리

모든 노조를 대표하는 하나의 교섭대표가 사용자와 교섭

⚙️ 진행 순서

①자율합의 → ②과반수노조 → ③공동대표단

⚠️ 예외

사용자 동의 시 개별교섭 가능

2. 제도 배경과 법적 근거

🗓️ 2011년 7월 1일 시행

복수노조 허용에 따른 교섭 혼란 방지 목적

  • •
    법적 근거: 노조법 제29조의2
  • •
    적용 대상: 모든 형태의 복수노조

3. 단일화 절차

1

교섭 요구

노조 → 사용자 (서면)

2

교섭요구 공고

사용자 (7일간)

3

참여노조 확정

사용자 공고 (5일간)

4

자율 결정

노조 간 협의 (14일)

5

과반수 통지

과반수 노조 → 사용자 → 공고(5일)

✓

대표노조 확정

이의 없으면 확정

📅 JS에너지지회 일정 예시 (터치하여 펼치기)

7/29~8/5: 교섭요구 및 공고

신규 기업노조 참여 가능 기간

8/6~8/11: 참여노조 확정

사용자 5일간 공고

8/12~8/25: 자율 결정

14일간 노조 간 협의

8/26~8/30: 과반수 공고

⚠️ 이의제기 가능 기간

4. 공정대표의무

⚖️ 소수노조 차별 금지!

교섭대표는 모든 노조를 공정하게 대표해야 함

✓ 의무자: 교섭대표노조 + 사용자
✓ 보호내용: 타임오프, 사무실, 조합비 등
✓ 구제: 3개월 내 노동위 신청

5. 실무 주요 쟁점

🔍 과반수 노조 판단 기준은?

해당 사업장 '종사근로자'만 계산

⚠️ 중국 파견직원은 제외!

직접 근로계약 관계가 없으면 조합원 수에서 제외

📝 이의제기는 어떻게?

1️⃣ 사용자의 과반수 공고 시작

2️⃣ 5일 이내 노동위원회 신청

3️⃣ 10일 내 노동위 결정

🚫 사용자 이익대표자란?
  • • 경영진 (대표이사, 임원 등)
  • • 인사/노무 담당자
  • • 실질적 인사권 보유자

6. 심화 학습

📋 교섭대표 확정 후 핵심사항

• 2년간 대표권한 보장

• 체결 단협은 모든 노조에 적용

• 불참 노조는 교섭권 상실

• 교섭 거부는 부당노동행위

⚠️ 현장 문제 대처법

공고 누락: 증거 확보 → 노동위 신고

어용노조 의심: 증거 수집 → 부당노동행위 제소

소수노조 차별: 공정대표의무 위반 시정신청
